Electronic Components

Arduino Nano + Expansion Board

The main microcontroller for your WebMob robot.

4 Servomotors (SG90 or similar)

These will control the movement of your robot.

Bluetooth Module (BLE 4.0 compatible)

Enables wireless communication with your smartphone.

Power Source (9V battery or similar)

Provides sufficient power for the Arduino and components.

Note: These are the basic components needed. Depending on the version of MobBob chosen, it could require some extra components.

Assembly

Put Everything Together

Assemble the 3D printed parts with the electronic components. Follow the instructions from the 3D model you've chosen.

Arduino Code

Upload the Code

Download and upload the Arduino code to your Nano board. I am using the code in the folder "MobBob-Control-Bluetooth".
